# Web Server Fingerprinting
## Returns Server Type and Server Version

 [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A simple web server fingerprinting tool that connects to the client through a TCP connection and finds out the server type and server version by parsing through a basic response returned by the server 

## Basic setup

Ensure python is installed and added to PATH

## Usage Instructions

Run server.py in one terminal, then run proj.py in another terminal for a demo understanding. 
For it to connect to actual global servers, change the server to the one desired in the targest address of proj.py along with the needed server port changes
